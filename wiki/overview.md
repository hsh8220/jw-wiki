# JW Wiki 개요

여호와의 증인 온라인 출판물(jw.org · wol.jw.org)의 내용을 수집해 만든 지식 베이스의 진입점.

## 탐색

- **[전체 카탈로그 →](../index.md)** — 모든 페이지 목록
- **소스** `sources/` — 수집한 기사·동영상별 요약
- **개념** `concepts/` — 교리·주제 (평화, 하느님의 왕국 …)
- **인물·장소** `entities/` — 예수, 사도, 길르앗 학교 …
- **성구** `scriptures/` — 인용된 성경 구절과 그 맥락 (이 위키의 핵심 연결 그래프)

## 현재 규모

- **동영상 연설 427편** 수집 (연사 50명):
  - 연설(StudioTalks) 102편
  - 아침 숭배(VODPgmEvtMorningWorship) 325편 — 제목 괄호의 핵심 성구를 `key_scripture`로 추출.
  - 전사본 415편(한국어 다수 + 일부 영어 폴백), 자막 미발행 12편은 메타데이터만.
- 성구 그래프: 62권의 성경책에서 인용된 **1,868개 구절**이 연설들과 연결됨
  (예: [[scriptures/시편]] 155구절·203회, [[scriptures/요한복음]]).
- 연사별 페이지에서 한 연사의 모든 연설을 연대순으로 볼 수 있음
  (예: [[entities/M. 스티븐 레트]] 24편, [[entities/제프리 W. 잭슨]] 25편).
- 반복 주제 개념 페이지 **30개**가 source 전체를 가로질러 연결됨.
